# Start OldDaisy Queue System

// turbo-all

1. Install dependencies (ถ้ายังไม่ได้ install):
```bash
cd /Users/home/.gemini/antigravity/scratch/photo-queue && npm install
```

2. Start the server:
```bash
cd /Users/home/.gemini/antigravity/scratch/photo-queue && node server.js
```

3. เปิดหน้าลูกค้า: http://localhost:3000
4. เปิดหน้าแอดมิน: http://localhost:3000/admin (PIN: 1234)
